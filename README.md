# 🤖 Personal AI Study Assistant

An AI-powered study tool built with Python and Groq API
that generates explanations, interview questions and quizzes
for any Gen AI topic.

## 🛠️ Built With
- Python
- Groq API (LLaMA 3.1 8B)
- Prompt Engineering

## ✨ Features
- Chain prompting for structured multi-step output
- JSON parsing for quiz generation
- Hallucination control with temperature=0
- System prompts for consistent AI behaviour

## 🚀 How To Run
1. Clone this repo
2. Run: pip install groq
3. Add your Groq API key in main.py
4. Run: python main.py
